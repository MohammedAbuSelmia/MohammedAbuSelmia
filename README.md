### 6.  Credit Card Fraud Detection — ML Pipeline
Tools: Python · scikit-learn · SQL Server · SQLAlchemy · joblib

An end-to-end machine learning pipeline built on 284,807 real credit card transactions to detect fraudulent activity — from SQL Server ingestion all the way to a saved, production-ready model.

What I built:
- Full data pipeline: SQL Server → Python cleaning → ML training → saved .pkl model
- Feature engineering: converted the Time column from elapsed seconds into a 24-hour clock format, revealing that most fraud occurs between midnight and 6 AM
- Handled severe class imbalance (only 0.17% of transactions are fraud) using RandomOverSampler
- Trained and compared two models: Logistic Regression and Random Forest
- Selected Logistic Regression as the best model — Recall of 90.7%, catching 128 out of 141 real fraud cases
- Saved the final model and scaler using joblib for future deployment without retraining

View Project: https://github.com/MohammedAbuSelmia/Credit-Card-Fraud-Detection
